# Путешествие по России

## Описание

Проектное задание на курсе "Веб Разработчик" от Яндекс Практикум.
В данной работе  применяются изученные технологии с 3 спринта по адаптивной верстке, а так же с прошлых спринтов.

Таких как:

+ Flexbox
+ Grid layout
+ Адаптивная верстка
+ БЭМ
